# Dev Log - DreamerHeroines

> 开发历程记录，用于个人档案/简历/自媒体素材

## 用法

每个 feature 完成后，用 2-3 分钟填写模板：

```markdown
## YYYY-MM-DD 功能名称

**Commit**: `abc1234`

**问题**: [为什么要做这个功能？遇到了什么问题？]

**尝试**:
1. 方案A ❌ [为什么失败]
2. 方案B ❌ [为什么失败]
3. 最终方案 ✅

**坑**: [开发过程中遇到的具体坑]

**收获**: [学到的东西，可以是技术点也可以是方法论]

**后续**: [还有什么要改进的？]
```

## 2026-03

### 2026-03-24 武器瞄准修正

**Commit**: `37b4208`

**问题**: 武器从玩家中心瞄准，子弹从枪口射出，导致精度偏差

**最终方案**: 将瞄准原点从玩家根节点改为枪口的世界坐标位置

**收获**: 在出现固定的瞄准方向偏差等异常行为时，注意角色根节点的位置

---

### 2026-03-24 准星 UI 系统

**Commit**: `926f779`

**问题**: 玩家射击时缺乏视觉反馈，不知道准星扩散状态

**实现**:
- CrosshairUI 场景，支持配置扩散、颜色、透明度
- 射击时准星扩散动画，恢复动画
- HUD 中继方法：spread, reload, ammo state
- weapon spread_changed 信号
