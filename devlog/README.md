# Dev Log - DreamerHeroines

> 开发历程记录，用于个人档案/简历/自媒体素材

## 用法

每个 feature 完成后，用 2-3 分钟填写模板：

```markdown
## YYYY-MM-DD 功能名称

**Commit**: `abc1234`

**问题**: [为什么要做这个功能？遇到了什么问题？]

**尝试**:
1. 方案A ❌ [为什么失败]
2. 方案B ❌ [为什么失败]
3. 最终方案 ✅

**坑**: [开发过程中遇到的具体坑]

**收获**: [学到的东西，可以是技术点也可以是方法论]

**后续**: [还有什么要改进的？]
```
