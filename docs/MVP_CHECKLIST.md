# Dreamer Heroines - MVP Checklist

> **项目**: 逐梦少女（Dreamer Heroines）2D横板射击游戏
> **引擎**: Godot 4.6.1  
> **版本**: MVP 1.0  
> **最后更新**: 2026-03-16

---

## 文档说明

本文档定义了项目的MVP（Minimum Viable Product，最小可行产品）验收标准。每个功能模块都有明确的预期效果、测试方法和验收标准。测试时应严格按照此文档进行验证，对于未达到预期的功能需记录并修复。

### 优先级说明

| 优先级 | 含义 | 说明 |
|--------|------|------|
| P0 | 核心功能 | 必须完成，否则游戏无法运行 |
| P1 | 重要功能 | 影响核心体验，应优先完成 |
| P2 | 次要功能 | 增强体验，可在MVP后完善 |
| P3 | 扩展功能 | MVP阶段可暂缓 |

### 验收状态标记

- [ ] 未实现
- [~] 部分实现
- [x] 已实现
- [!] 存在问题（需修复）

## 1. 玩家系统 (Player System)

### 1.1 基础移动

| 功能 | 优先级 | 预期效果 | 测试方法 | 状态 |
|------|--------|----------|----------|------|
| 左右移动 | P0 | 使用A/D或方向键可平滑左右移动，速度300px/s | 按键测试，观察移动流畅度 | [~] |
| 跳跃 | P0 | 空格/W键触发跳跃，高度约120px，有自然弧线 | 测试跳跃高度和距离 | [~] |
| 土狼时间 | P1 | 离开平台后0.1秒内仍可跳跃 | 在平台边缘测试跳跃时机 | [~] |
| 跳跃缓冲 | P1 | 落地前0.1秒内按跳跃，落地后立即跳跃 | 提前按键测试 | [~] |
| 可变跳跃高度 | P1 | 按住跳跃键跳得更高，提前松开跳得更低 | 对比长按和短按跳跃高度 | [~] |
| 二段跳 | P2 | 空中可再跳跃一次（需配置开启） | 空中再次按跳跃键 | [~] |
| 下蹲 | P1 | S键或下方向键下蹲，降低受击判定，速度减半 | 测试下蹲状态和移动速度 | [~] |
| 冲刺 | P1 | Shift键触发冲刺，速度提升50%，持续0.5秒，冷却2秒 | 测试冲刺距离和冷却 | [~] |

**当前实现状态**: 基础移动已实现，但冲刺无敌帧、冷却计时器需完善

**测试要点**:
1. 移动是否流畅无卡顿
2. 跳跃手感是否自然
3. 土狼时间和跳跃缓冲是否可感知
4. 冲刺是否有明显反馈

- [ ] 至少2种武器可用（步枪、霰弹枪）
- [ ] 至少3种敌人类型（近战、远程、飞行）

- [ ] 投射物系统正常工作

## 附录

### A. 配置文件说明

| 文件 | 用途 | 关键参数 |
|------|------|----------|
| `config/gameplay_params.json` | 游戏手感参数 | 移动速度、跳跃高度、无敌时间等 |
| `config/weapon_stats.json` | 武器数值 | 伤害、射速、弹匣容量等 |
| `config/enemy_stats.json` | 敌人数值 | 生命值、速度、伤害等 |
