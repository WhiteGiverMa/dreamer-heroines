# UI 模块

游戏用户界面脚本目录。

---

## 职责

- 游戏菜单系统（主菜单、暂停菜单、游戏结束）
- 游戏内 HUD 显示（生命值、弹药、分数、目标）
- 设置面板（分辨率、音量、灵敏度、窗口模式）
- 存档槽位管理组件
- 加载界面与进度显示
- 动态准星 UI 组件

---

## 文件

| 名称 | 用途 | 备注 |
|------|------|------|
| main_menu.gd | 主菜单逻辑 | **入口点**，处理开始游戏、存档加载、设置 |
| hud.gd | 游戏内 HUD | **入口点**，生命值/弹药/分数显示、伤害反馈 |
| pause_menu.gd | 暂停菜单 | **入口点**，暂停时继续/重启/设置/退出 |
| game_over.gd | 游戏结束界面 | 胜利/失败/超时状态，统计数据展示 |
| settings_panel.gd | 设置面板 | 分辨率/窗口模式/音量/灵敏度/VSync，可复用于主菜单和暂停菜单 |
| save_slot.gd | 存档槽位组件 | 显示单个存档状态，加载/删除操作 |
| loading_screen.gd | 加载界面 | 进度条、状态文本、淡出动画 |
| crosshair_ui.gd | 动态准星 | 扩散可视化、换弹/空弹匣状态颜色变化 |
| input_debug_widget.gd | 输入调试控件 | 实时显示跳跃/冲刺/移动/射击输入状态，仅调试用 |
